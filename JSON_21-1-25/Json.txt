JSON****
--JSON stands for Javascript object Notation
--JSON is the Javascript technique used to transfer the data between client to server and server to client.

SYNTAX:
JS object:-
***************
let a = {
        "name":"Sanyu",
        "age"=19
    }    

JSON object:-
***************
let a = {
        "name":"Sanyu",
        "age"=19
    } 

--For creation of the JSON file we need to create is as "Filename.json" 

JSON Methods
1. JSON.Stringify()-Used to convert the JS object to JSON Object
2. JSON.Parse()-Used to convert JSON object to JS object
